## Struktur

```

├── #hero
├── #ueber-uns
├── #leistungen

└── #kontakt

├── Geschichte

Leistungen (/leistungen)
├── Maßanfertigung (/leistungen/massanfertigung)

├── Umarbeitung (/leistungen/umarbeitung)

├── Ringe

├── Unikate

├── Formular
├── Karte
└── Terminbuchung
```

## URLs (Beispiel)

```
/                       → Startseite
/werkstatt              → Werkstatt
/leistungen             → Alle Leistungen (Übersicht)
/leistungen/trauringe   → Trauringe (eigene Seite)
/galerie                → Galerie
/massanfertigung        → Maßanfertigung (Conversion-Seite)
/kontakt                → Kontakt
/impressum              → Impressum
/datenschutz            → Datenschutz
```
