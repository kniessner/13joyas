# Joyas Design System

Welcome to the Joyas block-theme pattern library. Use this Storybook to view and iterate on patterns, blocks, and components before committing to WordPress.

## Philosophy

- Tokens live in `theme.json` and are synced to SCSS variables in `tokens.scss`.
- Patterns here mirror the WordPress block patterns found in `joyas-block-theme/patterns/`.
- Components are pure HTML/CSS/SCSS — no JavaScript frameworks required.
